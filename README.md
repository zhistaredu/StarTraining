<h2 align="center">职星学院企业培训系统</h2>

<a name="tQAEi"></a>
## 开源版介绍
一款可学习的企业培训系统，系统集成了用户管理、角色管理、部门管理、课件上传、课程创建、课程学习、课程统计等功能，学习流程完善。

<a name="tQAEi"></a>
## 商业版本
如果开源版本无法满足您的需求，或者有需求需要定制，可以考虑我们的商业版本<br />
**职星学院是一款专业的企业培训系统**，旨在助力企业构建高效、专业的团队。该系统提供了一站式、个性化的培训解决方案，以满足企业多样化的培训需求。<br />职星学院基于Java和MySQL进行开发，采用前后端分离的模式，前端核心框架为Vue3，后端核心框架为Spring Boot 2.5.8，最高可支持java17版本，确保了系统的稳定性和高效性。<br />
商业版官网地址：https://staredupro.cn/#/?from=github

<a name="tQAEi"></a>
## 技术栈
1、JDK >= 1.8 (推荐1.8版本)<br />2、Mysql >= 8.0 (推荐8.0版本)<br />3、Redis >= 3.0<br />4、Maven >= 3.0<br />5、SpringBoot2.5.8<br />6、VUE2<br />7、JavaScript
